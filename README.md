Hobby.ng Navigation Bar

A simple, responsive navigation bar built with HTML5 and CSS3 for the Hobby.ng landing page.

Features
Responsive navigation layout
Mobile-friendly design using CSS media queries
Custom CSS variables for easy theme customization
Hover effects for interactive links and buttons
Font Awesome icons integration

Technologies Used
HTML5
CSS3 (Flexbox & Media Queries)
Font Awesome
